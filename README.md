Library Management System — BIT1123 OOP Assignment 2
A Java console-based Library Management System for the BIT1123 Object Oriented Programming assignment

Features:

Add, display, search and delete Books and Magazines
Register and manage library members
Borrow and return items with validation
Color-coded console output using ANSI color codes

OOP Concepts Demonstrated:

Abstraction (LibraryItem abstract class)
Inheritance (Book and Magazine extend LibraryItem)
Polymorphism (method overriding, runtime dispatch)
Encapsulation (private attributes with getters/setters)
Classes and Objects
